# cleanCalc
a very basic JS calculator, built with the goal of having a very clean UI and very clean code.
a "demo" of it is up right now at https://www.nevin.cc/files/projects/cleanCalc/

You can embed it in your website/html doc very easily with just one line:
`<iframe src="https://www.nevin.cc/files/projects/cleanCalc/" title="cleanCalc" width="240" height="365" style="border: none;"></iframe>`
